# Risk Register Template

A simple risk register used during a diagnostic sprint and continued during delivery.

## Columns
1. Risk statement  
2. Impact if it happens  
3. Likelihood  
4. Severity  
5. Trigger signal  
6. Mitigation plan  
7. Owner  
8. Status  
9. Next review date  

## Operating rule
Review weekly. Close risks only when the trigger is no longer possible.
